## Troubleshooting Guide

### If Bonus Information Changes
1. Update `lib/fantasySports.js` with new bonus amount
2. Update sign-up steps if requirements changed
3. Update `bonusInfo` field with new terms
4. Run `npm run build` to verify
5. Redeploy to hosting

### If Platform Link Breaks
1. Check if platform is still operating
2. Verify affiliate program is still active
3. Update affiliate link if moved
4. Test link works before deployment
5. Add comment with verification date

### If Bonus Terms Are Confusing
- Add clarifying text in `bonusInfo` field
- Include mention of "terms apply" or "offer varies by state"
- Link to platform's terms page if needed
